Options

Options: { lowerCase?: boolean; paddedLength?: int; prefix?: string; radix?: Radix; separator?: string; suffix?: string }

The object with the following optional fields.

Type declaration

  • Optional lowerCase?: boolean

    Whether the formatted string is lowercase or not. The default is false.

  • Optional paddedLength?: int

    The length of the "0" padded formatted string for each byte. The default is determined by radix.

    radix default of paddedLength
    16 2
    10 3
    8 3
    2 8
  • Optional prefix?: string

    The prefix of the formatted string for each byte. The default is "".

  • Optional radix?: Radix

    The radix of the formatted string. 2, 8, 10, and 16 are available values. The default is 16.

  • Optional separator?: string

    The separator between the formatted strings of each byte. The default is "".

  • Optional suffix?: string

    The suffix of the formatted string for each byte. The default is "".

Radix

Radix: typeof _radixes[number]

2, 8, 10, or 16.
